# Online Renaissance Art Exhibition

## Giới thiệu

Website triển lãm tranh thời Phục Hưng, cung cấp carousel tác phẩm, phần giới thiệu, tìm kiếm hình ảnh và mục liên hệ.

## Thành viên nhóm

* Trần Quỳnh Anh
* Lý Đoàn Xuân Khiết

## Phân công công việc

* **Lý Đoàn Xuân Khiết:** Carousel, phần Giới thiệu, Liên hệ, Footer
* **Trần Quỳnh Anh:** Khung trang, Header, Navigation Bar, Tìm kiếm hình ảnh

## Cách chạy dự án

1. Clone repo:

   ```bash
   git clone (https://github.com/quynhengg/trienlamtranh.git)
   ```
2. Mở `index.html` bằng trình duyệt.
